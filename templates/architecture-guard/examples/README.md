# Architecture Guard Samples

These files are sample governance artifacts for the shipped
`architecture-guard` template.

They are examples for this repository, not live runtime configuration.

Included samples:

- `runtime-boundaries.json`: example architecture contract
- `runtime-boundaries-decommission.json`: example decommission ledger
- `runtime-boundaries-test-governance.json`: example test-governance artifact

For a real repository, prefer repo-local governance artifacts under:

- `.forge/governance/architecture-contract.json`
- `.forge/governance/decommission-ledger.json`
- `.forge/governance/test-governance.json`
